# Conscia Backend

NestJS backend for the Conscia mobile app.

## Setup

```bash
npm install
```

## Run

```bash
npm run start:dev
```

By default the API runs at:

```text
http://localhost:3000/api
```

## API Documentation

Open Swagger UI in your browser:

```text
http://localhost:3000/api/docs
```

Swagger UI lists the available endpoints, request parameters, request bodies, response schemas, and lets you test API calls directly from the browser. For protected endpoints, click **Authorize** and enter a JWT bearer token.

## Tests

```bash
npm run test
npm run test:e2e
```
